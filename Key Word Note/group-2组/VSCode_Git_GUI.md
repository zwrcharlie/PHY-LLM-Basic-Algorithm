# VSCode Git GUI（实验记录本可视化助手）

## 零基础解释
VSCode Git GUI就像**实验记录本的可视化助手**，让你不用记住复杂命令，通过点击操作完成实验记录管理。

## 物理场景类比解释
在高能物理实验中：
- Git GUI相当于**探测器数据管理控制台**：
  - 类似示波器的图形界面操作面板
  - 支持拖拽式实验数据版本切换
  - 图形化显示参数调整差异（如同对比波形变化）
  - 支持"Tag"标记关键实验时刻（如同标记重要观测事件）

## 物理场景类比
- Git GUI相当于**实验室的电子记录本界面**：
  - 类似探测器控制软件的可视化面板
  - 支持拖放式实验数据管理
  - 图形化显示实验参数变更

## 基本操作
```bash
# 运行环境：Windows 11 + VSCode GitLens插件(v1.0.0)
# 安装VSCode Git插件（推荐GitLens）
code --install-extension eamodio.gitlens  # 安装GitLens插件
# 或使用命令面板安装：
# Ctrl+Shift+P → 输入"Git: Install GitLens"
```

## 零基础使用教程
### 提交实验记录
1. 打开Git Changes侧边栏（如同实验记录本的修改追踪页）
2. 点击"+"号暂存所有修改（类似标记待归档的实验数据）
3. 输入中文提交信息（记录实验操作日志）
4. 点击"Commit"按钮（完成实验记录归档）

### 查看实验历史
1. 打开GitLens插件（扩展商店搜索"GitLens"）
2. 查看可视化分支图（如同不同实验方案的演进树）
3. 点击提交记录查看实验变更细节（类似翻阅实验记录本）
4. 右键文件选择"Git: Show File History"查看版本差异（如同对比实验数据修正）

## 物理系学生常见应用场景
- 可视化管理实验数据版本
- 图形化对比实验参数调整
- 快速查看实验记录历史

## VSCode操作指引
1. 安装GitLens插件（扩展商店搜索"GitLens"）
2. 在"Git Graph"视图查看分支结构（支持可视化合并操作）
3. 右键提交记录选择"Create Tag"标记重要实验版本（如关键参数确定时刻）
4. 使用"Compare"功能对比不同实验版本（如同对比实验条件变化）
5. 通过"Timeline"视图查看实验记录时间轴（类似实验日志时间戳）
6. 使用"Revert"功能回退错误实验数据（如同修正异常测量值）
