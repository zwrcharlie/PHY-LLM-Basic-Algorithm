## 一、研究背景与动机

### 1.1 主流方法的瓶颈

2017年之前，序列转换（sequence transduction）任务（如机器翻译、语言建模）的主流模型是基于**循环神经网络（RNN）**的编码器-解码器架构，包括 LSTM 和 GRU。这类模型存在根本性缺陷：

| 问题 | 描述 |
|------|------|
| **无法并行化** | RNN 按时间步顺序计算，每个隐藏状态 $h_t$ 依赖 $h_{t-1}$，训练时无法并行 |
| **长距离依赖难学习** | 序列越长，梯度消失/爆炸越严重，远距离位置间的依赖难以捕捉 |
| **内存瓶颈** | 长序列下批处理受内存限制 |

虽然注意力机制已被广泛用于辅助 RNN 建立长距离依赖，但几乎所有方案都将注意力与循环网络**配合使用**，而非替代它。

### 1.2 本文的核心主张

> **完全抛弃循环和卷积，仅依靠注意力机制构建序列转换模型。**

---

## 二、模型架构：Transformer

Transformer 遵循标准的**编码器-解码器**结构，但所有子层均基于注意力和全连接层，无任何循环或卷积单元。

### 2.1 总体架构

```
输入序列
   │
输入嵌入 + 位置编码
   │
┌──┴──────────────────────────┐
│     编码器（×N=6）           │
│  ┌─────────────────────┐   │
│  │ 多头自注意力          │   │
│  │ Add & Norm           │   │
│  │ 逐位置前馈网络        │   │
│  │ Add & Norm           │   │
│  └─────────────────────┘   │
└──┬──────────────────────────┘
   │ 编码器输出（K, V）
┌──┴──────────────────────────┐
│     解码器（×N=6）           │
│  ┌─────────────────────┐   │
│  │ 掩码多头自注意力      │   │
│  │ Add & Norm           │   │
│  │ 编码器-解码器注意力   │   │
│  │ Add & Norm           │   │
│  │ 逐位置前馈网络        │   │
│  │ Add & Norm           │   │
│  └─────────────────────┘   │
└──┬──────────────────────────┘
   │
线性层 + Softmax
   │
输出概率
```

### 2.2 编码器

- 由 **N=6** 个相同层堆叠而成
- 每层包含两个子层：
  1. **多头自注意力**（Multi-Head Self-Attention）
  2. **逐位置前馈网络**（Position-wise Feed-Forward Network）
- 每个子层均使用**残差连接**（residual connection）和**层归一化**（Layer Normalization）：
  ```
  输出 = LayerNorm(x + Sublayer(x))
  ```
- 所有子层及嵌入层的输出维度统一为 $d_{model} = 512$

### 2.3 解码器

- 同样由 **N=6** 个相同层堆叠而成
- 每层包含**三个**子层：
  1. **掩码多头自注意力**（Masked Multi-Head Self-Attention）——防止位置 $i$ 看到未来位置
  2. **编码器-解码器注意力**（Cross-Attention）——Query 来自解码器，Key/Value 来自编码器输出
  3. **逐位置前馈网络**

## 三、核心技术：注意力机制

### 3.1 缩放点积注意力（Scaled Dot-Product Attention）

**公式：**

$$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V$$

**各符号含义：**
- $Q$（Query）：查询矩阵，维度 $d_k$
- $K$（Key）：键矩阵，维度 $d_k$
- $V$（Value）：值矩阵，维度 $d_v$
- $\sqrt{d_k}$：缩放因子

**为什么要缩放？**

当 $d_k$ 较大时，$QK^T$ 的点积数值很大，会将 softmax 推入梯度极小的饱和区。除以 $\sqrt{d_k}$ 可将方差控制在合理范围内。

**与加性注意力的对比：**

| 类型 | 实现 | 速度 | 大 $d_k$ 下表现 |
|------|------|------|----------------|
| 点积注意力（本文）| 矩阵乘法 | 快，空间高效 | 需要缩放 |
| 加性注意力 | 单隐层前馈网络 | 慢 | 直接适用 |

### 3.2 多头注意力（Multi-Head Attention）

**核心思想：** 不用单个注意力函数处理全维度，而是将 Q、K、V 分别**线性投影**到 $h$ 个不同的低维子空间，在各子空间并行执行注意力，最后拼接并再次投影。

**公式：**
$$\text{MultiHead}(Q, K, V) = \text{Concat}(\text{head}_1, \ldots, \text{head}_h)W^O$$
$$\text{其中} \quad \text{head}_i = \text{Attention}(QW_i^Q, KW_i^K, VW_i^V)$$

**参数设置（base 模型）：**
- 头数 $h = 8$
- 每头维度 $d_k = d_v = d_{model}/h = 512/8 = 64$
- 总计算量与单头全维度注意力相当

**多头的优势：** 允许模型在不同表示子空间、不同位置上同时关注不同类型的信息（如一个头关注句法，另一个头关注语义）。

### 3.3 注意力在 Transformer 中的三种用途

| 用途 | Q 来源 | K/V 来源 | 作用 |
|------|--------|---------|------|
| 编码器自注意力 | 编码器上一层 | 编码器上一层 | 编码器每个位置可关注所有其他位置 |
| 解码器掩码自注意力 | 解码器上一层 | 解码器上一层（mask 未来）| 保持自回归性质 |
| 编码器-解码器注意力 | 解码器上一层 | 编码器输出 | 解码器关注完整输入序列 |

## 四、其他关键组件

### 4.1 逐位置前馈网络（FFN）

每层的注意力子层之后接一个全连接前馈网络，对每个位置**独立且相同地**应用：

$$\text{FFN}(x) = \max(0, xW_1 + b_1)W_2 + b_2$$

- 输入/输出维度：$d_{model} = 512$
- 中间层维度：$d_{ff} = 2048$
- 相当于两个核大小为 1 的卷积

### 4.2 位置编码（Positional Encoding）

由于 Transformer 无循环无卷积，天然不感知序列顺序，需要显式注入位置信息。

本文使用正弦/余弦函数编码：

$$PE_{(pos, 2i)} = \sin\left(\frac{pos}{10000^{2i/d_{model}}}\right)$$
$$PE_{(pos, 2i+1)} = \cos\left(\frac{pos}{10000^{2i/d_{model}}}\right)$$

**优点：**
- 波长从 $2\pi$ 到 $10000 \cdot 2\pi$ 构成等比数列，覆盖不同尺度的位置关系
- 对于固定偏移 $k$，$PE_{pos+k}$ 可以表示为 $PE_{pos}$ 的线性函数，便于模型学习相对位置
- 可能泛化到训练时未见过的更长序列

实验表明：正弦编码与学习型位置嵌入效果**几乎相同**（表3行E），选择正弦版本是因为其更好的泛化潜力。

### 4.3 嵌入与 Softmax

- 使用学习型嵌入将输入/输出 token 转换为 $d_{model}$ 维向量
- 输入嵌入、输出嵌入、Softmax 前线性层**共享同一权重矩阵**
- 嵌入层权重乘以 $\sqrt{d_{model}}$ 以平衡与位置编码的量级

## 五、为什么选择自注意力？

论文从三个维度对比自注意力、循环层和卷积层（表1）：

| 层类型 | 每层计算复杂度 | 顺序操作数 | 最大路径长度 |
|--------|---------------|-----------|-------------|
| **自注意力** | $O(n^2 \cdot d)$ | $O(1)$ | **$O(1)$** |
| 循环层（RNN）| $O(n \cdot d^2)$ | $O(n)$ | $O(n)$ |
| 卷积层 | $O(k \cdot n \cdot d^2)$ | $O(1)$ | $O(\log_k n)$ |
| 受限自注意力 | $O(r \cdot n \cdot d)$ | $O(1)$ | $O(n/r)$ |

（$n$=序列长度，$d$=表示维度，$k$=卷积核大小，$r$=受限邻域大小）

**关键优势：**

1. **最短路径长度 $O(1)$**：任意两个位置之间信号传播路径为常数，极大降低了长距离依赖的学习难度
2. **完全可并行**：顺序操作数为 $O(1)$，训练速度大幅提升
3. **当 $n < d$ 时计算效率更高**（NLP 中常见情形）
4. **可解释性**：注意力分布可直接可视化，不同头学到了不同的句法/语义模式

## 六、训练设置

### 6.1 数据集

| 任务 | 数据集 | 规模 | 词表 |
|------|--------|------|------|
| 英语→德语 | WMT 2014 EN-DE | 450万句对 | BPE，37000 token |
| 英语→法语 | WMT 2014 EN-FR | 3600万句对 | Word-piece，32000 token |

### 6.2 硬件与时间

| 模型 | GPU | 训练步数 | 训练时间 |
|------|-----|---------|---------|
| Base | 8× NVIDIA P100 | 100,000步 | **12小时** |
| Big | 8× NVIDIA P100 | 300,000步 | **3.5天** |

### 6.3 优化器

使用 **Adam** 优化器，$\beta_1=0.9$，$\beta_2=0.98$，$\epsilon=10^{-9}$。

**自适应学习率调度：**
$$lrate = d_{model}^{-0.5} \cdot \min(step\_num^{-0.5},\ step\_num \cdot warmup\_steps^{-1.5})$$

- 前 $warmup\_steps=4000$ 步线性增大学习率
- 之后按步数的平方根反比衰减

### 6.4 正则化

| 方法 | 细节 |
|------|------|
| 残差 Dropout | 每个子层输出应用，Base 模型 $P_{drop}=0.1$ |
| 嵌入 Dropout | 嵌入+位置编码之和上应用 |
| 标签平滑 | $\epsilon_{ls}=0.1$，虽增大困惑度，但提升 BLEU 和准确率 |

## 七、实验结果

### 7.1 机器翻译（表2）

| 模型 | EN-DE BLEU | EN-FR BLEU | 训练成本（FLOPs）|
|------|-----------|-----------|----------------|
| GNMT + RL [31] | 24.6 | 39.92 | $2.3\times10^{19}$ |
| ConvS2S [8] | 25.16 | 40.46 | $9.6\times10^{18}$ |
| MoE [26] | 26.03 | 40.56 | $2.0\times10^{19}$ |
| ConvS2S Ensemble | 26.36 | 41.29 | $7.7\times10^{19}$ |
| **Transformer (base)** | **27.3** | **38.1** | **$3.3\times10^{18}$** |
| **Transformer (big)** | **28.4** | **41.0** | $2.3\times10^{19}$ |

**亮点：**
- EN-DE：Transformer (big) 比此前最优（含集成模型）高出 **2+ BLEU**，达到 28.4 新 SOTA
- EN-FR：Transformer (big) 达到 41.0 BLEU，以**不足之前最优模型 1/4 的训练成本**超越所有单模型
- Transformer (base) 训练仅 12 小时，BLEU 27.3，已超越所有此前发布的单模型和集成模型

### 7.2 架构消融实验（表3，EN-DE newstest2013）

**（A）注意力头数量的影响**

| 头数 $h$ | $d_k=d_v$ | BLEU |
|---------|-----------|------|
| 1 | 512 | 24.9 |
| 4 | 128 | 25.5 |
| **8** | **64** | **25.8** |
| 16 | 32 | 25.8 |
| 32 | 16 | 25.4 |

> 8头为最优；头太少或太多都会下降。单头比最优设置低 0.9 BLEU。

**（B）注意力键维度 $d_k$ 的影响**

- 减小 $d_k$ 会损害模型质量，说明判断 query-key 相容性不简单，点积可能不是最优的兼容函数。

**（C）模型规模的影响**

- $d_{model}$、$d_{ff}$、层数 $N$ 增大均带来性能提升

**（D）正则化的影响**

- Dropout 和标签平滑均有正向效果

**（E）位置编码类型**

- 正弦编码 vs 学习型嵌入：几乎无差异（25.8 vs 25.7）

## 八、与自注意力动机相关的深层分析

### 8.1 长距离依赖的信号传播

- **RNN**：位置 $i$ 和位置 $j$ 之间的信号需经过 $|i-j|$ 步，梯度随距离指数衰减
- **卷积**：需要 $O(n/k)$ 层才能连接所有位置对
- **自注意力**：任意两位置之间恒为**常数步**，长距离依赖信号不衰减

### 8.2 可解释性

各注意力头学到了不同的行为：
- 部分头专门关注句法结构（如主谓关系）
- 部分头关注语义相关性
- 各头之间存在明显的功能分工

## 九、局限性与未来方向

论文结尾提出了若干未来研究方向：

1. **扩展到文本以外的模态**：图像、音频、视频等
2. **局部受限注意力**：处理极长序列时降低 $O(n^2)$ 复杂度
3. **减少生成的顺序依赖性**：探索更并行的生成方式

## 十、历史影响与总结

### 主要贡献

1. **提出 Transformer**：第一个完全基于注意力机制的序列转换模型，彻底去除循环和卷积
2. **缩放点积注意力**：简单高效，可用矩阵乘法高速实现
3. **多头注意力**：允许模型在多个子空间同时关注不同类型的信息
4. **正弦位置编码**：无参数、可泛化的位置感知方案
5. **在机器翻译上创新 SOTA**：以极低训练成本大幅超越此前最优

### 深远影响

> **一句话总结：** 本文提出的 Transformer 架构证明，仅靠注意力机制（无循环无卷积）即可在序列转换任务上实现更高质量、更快训练、更强并行性，从根本上改变了 NLP 乃至整个深度学习领域的技术路线，成为此后几乎所有大语言模型的基础架构。
